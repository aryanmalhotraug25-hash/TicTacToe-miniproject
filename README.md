#  Tic-Tac-Toe Mini Project

A simple and interactive Tic-Tac-Toe game built using **HTML5**, **CSS3**, and **JavaScript**.  
This project demonstrates core web development concepts such as DOM manipulation, event handling, and responsive design.

---

##  Features
- Classic **3x3 Tic-Tac-Toe** board
- **Two Game Modes:**
  -**Human vs Human** → Two players take turns on the same device
  - **Human vs Computer** → Play against a basic AI opponent
- Detects win, draw, and invalid moves
- Reset/Restart game functionality
- Responsive design for desktop and mobile
- Clean and minimal UI with CSS styling

---

## Technologies Used
- **HTML5** → Structure of the game board
- **CSS3** → Styling and layout
- **JavaScript (ES6)** → Game logic and interactivity
